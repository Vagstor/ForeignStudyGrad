Задача: сделать простую систему мониторинга. Это веб-прилоежиен, в которое если зайти ,то сразу видно список приложений с лампочками "красное/зеленое" означающее "работает/не работает".
Первая версия будет простой и иметь массу ограничений дальше будем ограничения снимать, а систему улучшать.

Для первой версии проверяем только HTTP-приложения путем отправки HTTP-запроса и ожидания в ответ подходящего HTTP-статуса. 

Для пользователя у приложений две части:
- админка
- собственно список приложений со статусами. 

Для начала вход в админку свободный без какой-либо проверки прав, для каждого приложения в админке задается только "Адрес, ожидаемые коды (по умолчанию - 200)".

Технически:
1. веб-часть: ASP.NET Core, bootstrap css
2. Бэкенд: PostgreSQL, linq2db


Будущие расширения:
- ведение истории  ошибок и просмотр лога когда была недоступность
- мониторинг не только HTTP, но и баз данных.
- email/web уведомления при сбоях
- проверка прав для входа в админку, создание и управление пользователями
